Score 0–10 = qualidade × fama × bônus de desconto.
Qualidade = limite inferior de Wilson (95%) das reviews positivas — junta "% positivas" e "nº de reviews" com confiança estatística (95% de 200 reviews vale menos que 95% de 200 mil). Fama (log das reviews) e desconto são modificadores suaves, então um bom jogo nunca é zerado por ter pouco desconto, nem um jogo ruim é salvo por ter muito.
A lista mostra por padrão os jogos com score 6–10; o restante fica num "Ver mais". Wishlist comparada via endpoint público da Steam (sem API key); biblioteca (jogos que você já tem) é opcional e usa uma API key só nesta requisição — nada é armazenado no servidor.
